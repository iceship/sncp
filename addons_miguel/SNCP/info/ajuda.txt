CRIAR RELAÇÕES ENTRE FATURAS, COMPROMISSOS, ORDENS DE COMPRA
0) A CATEGORIA ASSOCIADA AO PRODUTO TEM DE TER CONTA DE AVALIAÇÃO DE STOCK DEFINIDA
1) COTAÇÕES
   1.1) CRIAR
   1.2) SEPARADOR INCOMING SHIPMENTS & INVOICES
        1.2.1) Controlo de faturação -> Based on incoming shipments
   1.3) PREENCHER LINHAS DA ORDEM
   1.4) DEPOIS DE PREENCHER OS CAMPOS GUARDAR
2) APROVAÇÕES DE ORDENS DE COMPRA
    2.1) CRIAR CABIMENTO E COMPROMISSO
    2.2) CRIAR
      2.2.1) PREENCHER OS CAMPOS
    2.3) CONTINUAR

4) CONTROLO DAS FATURAS
 4.1) ON INCOMING SHIPMENTS
  4.2) SELECIONAR O ASSOCIADO A ORDEM DE COMPRA APROVADA(RETIRAR OS FILTROS)
   4.3) ABRIR E CLICAR NO BOTÃO RECEIVE (CLICAR NO BOTÃO RECEIVE)
    4.4) CLICAR NO BOTÃO CREATE INVOICE/REFUND (PREENCHER OS CAMPOS OBRIGATÓRIOS)


PARA COPIAR A INFORMAÇÂO DE UMA BASE DE DADOS PARA OUTRA NO PGADMIN
1. EXECUTAR QUERY "CREATE EXTENSION dblink"
2. USAR A FUNÇÃO DBLINK PARA FAZER CONEXÃO REMOTA E EXECUTAR QUERY NA TABELA DE UMA OUTRA BASE DE DADOS,
COM ESTA NO MESMO SERVIDOR:
        dblink('dbname=NOME_bd', QUERY)
